How to install Python on Fedora 43

Hey there, fellow Fedora enthusiasts! Ready to dive into the world of Python on the blazing-fast Fedora 43? Installing Python is easier than brewing a good cup of coffee, and this guide will walk you through it step-by-step. Whether you're a seasoned coder or just starting your programming journey, let's get Python up and running on your system.

Why Python on Fedora?

Fedora is a fantastic platform for development, and Python is a versatile language perfect for everything from scripting to AI. By combining the two, you're setting yourself up for some serious coding fun. Plus, Fedora keeps its packages up-to-date, so you'll always be running a stable and secure version of Python.

Checking if Python is Already Installed

Before we install, let's check if Python is already on your system. Fedora often comes with Python pre-installed! Open your terminal and type:

python3 --version

If you see a version number (e.g., Python 3.12.x), then Python is already installed, and you're good to go! You can skip ahead to the "Verifying pip" section below. If not, let's proceed with the installation.

Installing Python on Fedora 43

The easiest way to install Python on Fedora is using the DNF package manager. Here's how:

  1. Open your terminal.
  2. Type the following command and press Enter:
    3. sudo dnf install python3
  3. You'll likely be prompted for your password. Enter it and press Enter.
  4. DNF will then download and install Python and any necessary dependencies. When prompted, type 'y' and press Enter to confirm the installation.

That's it! Python is now installed.

Verifying the Installation

To confirm that Python has been installed correctly, run the following command in your terminal:

python3 --version

You should now see the Python version number displayed. Congratulations, you've successfully installed Python!

Installing pip (Python Package Installer)

pip is the package installer for Python, and it's essential for installing libraries and modules. It might be installed along with Python, but let's make sure. Run this command:

pip3 --version

If you see a version number, you're all set. If you get an error, install pip with:

sudo dnf install python3-pip

Once installed, verify pip with the command above again.

Example: Installing a Python Package (NumPy)

Let's install a popular Python package called NumPy, which is used for numerical computing. This will also test that your pip installation is working correctly.

In your terminal, type:

pip3 install numpy

After the installation completes, you can test it by opening the Python interpreter and importing NumPy:

  1. Type python3 in your terminal to start the Python interpreter.
  2. Type import numpy as np and press Enter.
  3. If no errors occur, NumPy has been successfully installed!
  4. Type exit() to exit the interpreter.

Wrapping Up

You've successfully installed Python and pip on your Fedora 43 system. Now you're ready to explore the vast world of Python programming. Happy coding!
