How to install Python on Fedora 43

Hey Fedora fans! Ready to get your Python on? Fedora 43 makes it super easy to install Python, letting you dive into scripting, automation, and all sorts of cool projects. Let's walk through the process step-by-step – it's easier than you think!

Checking if Python is Already Installed

First things first, let's check if Python is already lurking on your system. Fedora often comes with Python pre-installed, but it's good to double-check. Open your terminal (you can usually find it in your applications menu, or press Ctrl+Alt+T) and type:

python3 --version

If you see a version number pop up (like "Python 3.12.0"), congratulations! You've already got Python. If you get an error message, don't worry, we'll install it next.

Installing Python 3 on Fedora 43

Fedora uses the DNF package manager, which makes installing software a breeze. To install Python 3, run the following command in your terminal:

sudo dnf install python3

You'll probably be prompted for your password – enter it and press Enter. DNF will then download and install Python and any necessary dependencies. It might ask you to confirm the installation by typing 'y' and pressing Enter.

Verifying the Installation

Once the installation is complete, let's verify that Python 3 is installed correctly. Run the version check again:

python3 --version

You should now see the Python 3 version number. Great job!

(Optional) Installing `pip` - The Python Package Installer

pip is a crucial tool for installing and managing Python packages. It's like the app store for Python code. While Python sometimes includes it, you might need to install it separately. Here's how:

sudo dnf install python3-pip

Again, you might need to confirm the installation with 'y'.

To verify pip is installed, run:

pip3 --version

Getting Started with Python

Now that you have Python and pip installed, you're ready to start coding! You can open a Python interactive shell by typing python3 in your terminal. You'll see a prompt like >>>, where you can start typing Python commands. For example:

>>> print("Hello, Fedora!")

Press Enter, and you should see "Hello, Fedora!" printed to the terminal.

Recap: The steps we took

  • Checked if Python was pre-installed.
  • Installed Python 3 using dnf.
  • Verified the installation.
  • Installed pip for package management (optional, but highly recommended!).

That's it! You're now equipped with Python on Fedora 43. Happy coding!
