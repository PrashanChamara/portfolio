How to Start with Snaps: Installing & Using Snap Package Manager on Linux

Published on: 2025/01/31 | Categories: Linux, Package Manager

Snaps are a modern packaging format designed to simplify software distribution across different Linux distributions. By using snaps, you can easily install thousands of applications, keep them up-to-date automatically, and avoid dependency conflicts.

What Are Snaps?

Snaps are self-contained software packages that include all their required dependencies. This means you can run them on most major Linux distributions without worrying about missing libraries. The Snap Store hosts a vast library of apps, from developer tools to desktop applications and server software.

1. Check If Snap Is Installed

Many modern Linux distros come with snapd (the Snap daemon) pre-installed. To see if it’s already on your system, open a terminal and run: 

snap version

If you see a version number for snapd, then you’re all set. Otherwise, you’ll need to install snapd.

2. Install Snapd on Your Linux Distribution

The method of installing snapd varies by distro. Here are a few examples:

  • Ubuntu / Debian: sudo apt update && sudo apt install snapd
  • Fedora: sudo dnf install snapd
  • Arch Linux: sudo pacman -S snapd

3. Installing Software via Snap

Once snapd is running, you can search for and install software from the Snap Store. For example, to search for VLC: 

snap find vlc

Then install it with:

sudo snap install vlc

Snaps update automatically, so you’ll always have the latest version of your installed apps.

4. Managing Snaps

Here are some common commands for working with snaps:

  • List installed snaps: snap list
  • Remove a snap: sudo snap remove <snap-name>
  • Refresh snaps: sudo snap refresh

Snaps are sandboxed for security, meaning each snap runs in a confined environment to minimize potential risks to your system.

Conclusion

Snaps offer an easy, secure way to install and manage software on your Linux system. By following these steps, you can install snapd, explore the Snap Store, and enjoy hassle-free software management. Whether you’re a developer or a daily user, snaps provide a powerful, consistent experience across Linux distributions.
